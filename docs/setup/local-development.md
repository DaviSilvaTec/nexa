# DESENVOLVIMENTO LOCAL

## FUNÇÃO DESTE ARQUIVO
Orientar a preparação do ambiente local para desenvolvimento.

## QUANDO CONSULTAR
Consultar este arquivo ao iniciar o projeto localmente ou preparar uma máquina nova.

## RELAÇÃO COM OUTROS ARQUIVOS
- Complementa `environment.md`.
- Depende de `../architecture/backend.md`.

## OBJETIVO
Padronizar o ambiente inicial para:
- rodar backend localmente;
- testar webhook;
- validar integrações;
- evoluir o projeto com previsibilidade.
